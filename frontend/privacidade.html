Política de Privacidade

Última atualização: abril de 2025

1. Sobre este sistema

O Rastreamento de Leads é uma plataforma de CRM (Customer Relationship Management) voltada para empresas que gerenciam contatos e comunicações via WhatsApp e Meta Ads. Este sistema é utilizado internamente por equipes de vendas e atendimento.

2. Dados coletados

O sistema coleta e armazena os seguintes dados de contatos (leads):

  • Nome e número de telefone (WhatsApp)
  • Endereço de e-mail (quando fornecido voluntariamente)
  • Histórico de interações e mensagens trocadas via WhatsApp
  • Origem do contato (anúncio, campanha ou indicação)
  • Status e etapa no funil de vendas

3. Como os dados são coletados

Os dados são coletados de duas formas:

  • WhatsApp Business API (Meta): mensagens enviadas voluntariamente pelo usuário para números de negócios cadastrados na plataforma.
  • Cadastro manual: inserção direta por membros da equipe comercial.

4. Finalidade do uso dos dados

Os dados coletados são utilizados exclusivamente para:

  • Gerenciar o relacionamento comercial com o contato
  • Registrar o histórico de atendimentos e comunicações
  • Mensurar a eficiência de campanhas de marketing
  • Facilitar o acompanhamento do processo de vendas

5. Compartilhamento de dados

Os dados não são compartilhados, vendidos ou cedidos a terceiros. O acesso é restrito aos usuários autorizados da empresa contratante do sistema.

6. Armazenamento e segurança

Os dados são armazenados em servidores seguros com acesso protegido por autenticação. Utilizamos boas práticas de segurança, incluindo senhas criptografadas e tokens de acesso com expiração.

7. Retenção dos dados

Os dados são mantidos pelo período necessário à prestação do serviço comercial. O contato pode solicitar a exclusão dos seus dados a qualquer momento.

8. Direitos do titular

Em conformidade com a LGPD (Lei Geral de Proteção de Dados — Lei nº 13.709/2018), o titular dos dados tem direito a:

  • Confirmar a existência de tratamento de seus dados
  • Acessar seus dados armazenados
  • Solicitar a correção ou exclusão dos dados
  • Revogar o consentimento a qualquer momento

9. Integração com Meta / WhatsApp

Este sistema utiliza a API oficial do WhatsApp Business fornecida pela Meta Platforms. O uso está em conformidade com os Termos de Serviço do WhatsApp Business e as Políticas da Plataforma Meta.

10. Contato

Para dúvidas, solicitações ou exercício dos seus direitos, entre em contato pelo e-mail registrado na empresa responsável pelo sistema.
